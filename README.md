# Login & Register Form UI

This is a simple and responsive Login and Registration Form UI created using HTML and CSS.
The project focuses on clean design and basic front-end structure for beginners.

## Features
- Login page UI
- Register page UI
- Responsive design
- Clean HTML and CSS
- Beginner friendly
